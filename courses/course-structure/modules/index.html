Modules

All materials for these modules will be shared through the Blackboard course site.

  • Module 1: What tools can we use for reproducible workflows?
    • Topics: GitHub and Markdown; RStudio and RMarkdown; Git
  • Module 2: How can we quickly visualize information to get a sense of what is going on?
    • Topics: Grammar of graphics intro; ggplot2; geoms; colors; titles
  • Module 3: What are the foundational building blocks of coding in R?
    • Topics: Objects; vectors; functions
  • Module 4: How do we isolate information from within a dataset?
    • Topics: selecting columns; filter rows; pipes; creating variables
  • Module 5: How do we summarize data and present these summaries in a human-readable format?
    • Topics: Summarizing; groups; restructuring
  • Module 6: How do we combine multiple sources of information?
    • Topics: Relational data; joins
  • Module 7: How do we efficiently work with special variable types?
    • Topics: Ordering characters; date-times; parsing strings
  • Module 8: Return of the plot… How can we make our data visualizations really “pop”?
    • Topics: Scales; color pallets; facets; combining graphical objects
  • Module 9: How do we create our own R functions?
    • Topics: Functions for humans; conditional execution; function arguments; web scraping
  • Module 10: How do we apply functions to multiple items?
    • Topics: Loops; functionals; multiple iteration
  • Module 11: How do we use functional programming for statistical analyses?
    • Topics: linear models; nested data; statistical simulations; resampling methods
  • Module 12: How do we create interactive data products?
    • Topics: Shiny apps
